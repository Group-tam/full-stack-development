Event Planning & Management System - Guide on how to run the application
1. Make sure that MongoDB and Node is running on your system
2. Go to the frontend directory of the project
3. Run npm run build on the terminal (or yarn run build if you have yarn)
4. Go to the root directory of the project
5. Run node . or node --experimental-strip-types . on the terminal if you are on Node version is between 22.6.00 and 23
6. To stop the server press Ctrl + C on the terminal.
