Event Planning & Management System - Guide on how to run the application
1. Make sure that MongoDB and Node is running on your system
2. Run npm install for frontend directory and root directory
3. Go to the frontend directory of the project
4. Run npm run build on the terminal (or yarn run build if you have yarn)
5. Go to the root directory of the project
6. Run node . or node --experimental-strip-types . on the terminal if you are on Node version is between 22.6.00 and 23
7. To stop the server press Ctrl + C on the terminal.

For MongoDB installation
1. Go to https://www.mongodb.com/try/download/community
2. Choose msi package and download
3. After download finished, open the installer to get to the setup wizard
4. Choose "Complete" as a setup type if prompted
5. For Service Configuration part, tick "Install MongoD as a Service" and choose "Run service as Network Service user"
6. Proceed with the installation (You can optionally install MongoDB Compass)
Note:
1. You can optionally install MongoDB shell for debugging purposes in https://www.mongodb.com/try/download/shell
2. The "MongoDB Database Server" should runs in the background after the installation (you can check it via the task manager)
3. On cmd (run as Adminstrator) You can use the command "net start MongoDB" to manually start the server,
"net stop MongoDB" to stop the server
